# 第一阶段：数学基础补强

## 阶段目标
掌握AI/机器学习所需的核心数学知识，为后续学习打下坚实基础。重点不是成为数学家，而是建立直觉，理解公式背后的几何意义和物理意义，并能用代码实现。

## 学习时间规划
*   **总时长**： 2-3个月
*   **每周投入**： 15-20小时
*   **建议节奏**： 每个子模块2-3周

## 阶段验收与总结

### 验收标准

#### 1. 理论理解 (笔试自测)
*   **线性代数**：能够手动计算简单的矩阵乘法、特征值；理解矩阵变换的几何意义。
*   **微积分**：能够计算多元函数的偏导数和梯度；能够推导梯度下降公式。
*   **概率统计**：能够应用贝叶斯定理解决实际概率问题；理解假设检验的基本流程。

#### 2. 编程能力 (代码实战)
*   **NumPy基础**：熟练使用NumPy进行向量化计算，避免使用循环。
*   **算法实现**：能够从零实现梯度下降算法优化简单函数。
*   **数据思维**：能够用统计量描述数据特征，并发现数据中的异常。

## 常见问题 (Q&A)

**Q1: 数学基础要学到什么程度？**
**A**: 理解概念和直觉比死记公式更重要。重点理解：
1.  为什么需要这个数学工具？
2.  在AI中的什么场景使用？
3.  如何用代码实现？

**Q2: 如果某个概念不理解怎么办？**
**A**:
1.  先看直观解释（推荐 3Blue1Brown, StatQuest）。
2.  再看数学推导。
3.  动手编程实现。
4.  不要卡太久，可以标记后续回看。

**Q3: 需要做完所有习题吗？**
**A**: 不必全部完成，但每种类型至少做2-3道典型题，确保掌握解题思路。
