# 模块四：循环神经网络 (RNN)

用于处理序列数据（文本、时间序列、语音）。

## 4.1 基础 RNN
*   **原理**: 当前时刻的输出依赖于当前输入和上一时刻的隐藏状态。
*   **问题**: 梯度消失/爆炸，无法捕捉长距离依赖。

## 4.2 LSTM (长短期记忆网络) ⭐⭐⭐⭐⭐
通过引入 **门控机制 (Gating)** 解决了长距离依赖问题。
*   **遗忘门 (Forget Gate)**: 决定丢弃什么信息。
*   **输入门 (Input Gate)**: 决定更新什么信息。
*   **输出门 (Output Gate)**: 决定输出什么信息。
*   **细胞状态 (Cell State)**: 贯穿整个序列的“高速公路”。

```python
nn.LSTM(input_size=10, hidden_size=20, num_layers=2, batch_first=True)
```

## 4.3 GRU (门控循环单元)
LSTM 的简化版，只有重置门和更新门，参数更少，训练更快，效果通常相当。

## 4.4 应用场景
*   **文本分类**: 情感分析。
*   **序列标注**: 命名实体识别。
*   **机器翻译**: Seq2Seq 模型。
*   **时间序列预测**: 股票、天气。
