# 5.6 综合项目与总结

## 第五阶段综合项目

### 项目1：智能问答系统（RAG）
**目标**：基于企业文档的智能问答系统
**功能**：
1. 上传文档（PDF、TXT、MD）
2. 自动构建向量库
3. 智能问答
4. 答案溯源（显示参考文档片段）

### 项目2：客服机器人（Agent）
**目标**：智能客服机器人
**功能**：
1. 意图识别（咨询、投诉、查询）
2. 多轮对话（维护上下文）
3. 知识查询（RAG）
4. 工单创建（调用外部API）

### 项目3：内容生成助手
**目标**：博客文章生成助手
**功能**：
1. 主题研究（联网搜索）
2. 大纲生成
3. 内容创作
4. SEO优化建议
