# 6.6 综合项目与总结

## 第六阶段综合项目

### 项目：企业级智能客服系统

**技术栈**：
- **后端**：Python (FastAPI) + Java (Spring Boot)
- **前端**：Vue.js
- **AI模型**：LLM + RAG
- **部署**：Docker + Kubernetes
- **监控**：Prometheus + Grafana

**核心功能**：
1. **智能问答**：基于RAG的企业知识库问答。
2. **多轮对话**：维护用户会话上下文。
3. **工单创建**：Agent自动调用Java后端接口创建工单。
4. **人工转接**：情感分析识别愤怒情绪，自动转接人工。
5. **数据分析**：Dashboard展示问答统计和热门问题。
